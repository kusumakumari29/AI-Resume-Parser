# AI Resume Parser

## Description
This project is developed using Python, Flask, SQLite and pdfplumber. It extracts candidate details from PDF resumes and stores them in a database.

## Features
- Upload PDF Resume
- Extract Name
- Extract Email
- Extract Phone Number
- Extract Skills
- Store Data in SQLite Database
- View All Candidates
- Search Candidates by Skill
- Delete Candidate Records
- Export Data to CSV

## Technologies Used
- Python
- Flask
- SQLite
- pdfplumber
- Bootstrap

## How to Run
1. Install the required packages:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open:
   http://127.0.0.1:5000
